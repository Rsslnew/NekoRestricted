**Powerful Telegram bot to download restricted content from private & public channels.**

## ✨ Features

- 📥 Download restricted media (video, photo, audio, document)
- 🔐 Dual login mode (User + Admin)
- 📦 Batch download support
- 📊 Real-time progress bar
- ❌ Cancel download anytime
- 💎 Premium system
- 🎨 Custom caption & thumbnail

---

# 1. Clone repository
git clone https://github.com/Rsslnew/AcxNekoBot.git
cd AcxNekoBot

# 2. Create virtual environment
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Edit configuration
nano config.env

# 5. Create directories
mkdir -p downloads thumbnails logs

# 6. Run bot
python AcxNeko.py

### 2. Deploy

Docker

docker compose up --build -d

Manual

pip install -r requirements.txt
python AcxNeko.py

## 📝 Commands

| Command | Description |
|---------|-------------|
| /start | Start bot |
| /dl link | Download post |
| /bdl start end | Batch download |
| /login | Login account |
| /help | Full help |
